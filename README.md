# Labguide 1 — Computer Conversations

Hands-on **Computer Conversations** lab material for DevRev: building a knowledge base and using an **agentic AI** (Computer) to answer support-style questions with human-like responses. The story follows **ACME**, an insurance broker, as they move from rigid workflows toward AI-assisted answers for internal IT and insurance inquiries.

## What’s in this repo

- **`docs/`** — Markdown lab pages (screenshots under `docs/images/`).
- **`mkdocs.yml`** — [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) site config.

Modules cover **lab registration and setup**, an **introduction to agentic workflows**, **Parts and Trials**, **KB articles** (manual authoring, **website scraping**, **AirSync with Confluence**), and **testing the Computer agent** with simple and complex questions.

## Local preview

```bash
pip install mkdocs-material
mkdocs serve
```

Open the URL shown in the terminal (e.g. `http://127.0.0.1:8000`).
